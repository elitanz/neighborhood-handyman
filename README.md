# Neighborhood Handyman — website

A custom one-page website for **Neighborhood Handyman LLC** (owner Jeff), a family-run
remodeling & repair service across the St. Paul / Maple Grove / Twin Cities metro —
kitchens, baths, basements, flooring, tile, trim, woodwork and home repairs.

Built from real public data (Nextdoor business page, 101 neighbor recommendations).
Features a scroll-scrub video hero (real woodworking footage played frame-by-frame on
scroll), services, project gallery, process, reviews, and a service-area map.

- **Live:** https://elitanz.github.io/neighborhood-handyman/
- Static HTML/CSS/JS — no build step. Just open `index.html`.
- Craftsman identity: wood + charcoal + amber palette, tape-measure ruler motif,
  Archivo + Space Mono type.

*Independent sample site. Not affiliated with the business.*
